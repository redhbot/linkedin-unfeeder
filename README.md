# LinkedIn Unfeeder

A Firefox extension that removes the LinkedIn feed so you can focus on what matters.

## Features

- Hides the main feed, "Start a post" box, news sidebar, and "new posts" pill
- Toolbar toggle to enable/disable without going to extension management
- State persists across browser restarts

## Install

1. Open `about:debugging#/runtime/this-firefox` in Firefox
2. Click **Load Temporary Add-on...**
3. Select `manifest.json` from this directory

## Usage

Click the extension icon in the toolbar to toggle the feed on or off.
